# PHARMACY-DATA-ANALYSIS
Eploratory data analysis of pharmaceutical sales data to uncover customer behavior, product performance using Excel analysis tool.

## Project Overview
This project analyzes pharmacy data to identify sales trends,medication demand pattern, inventory performance and business insights that can support data driven decision making.

## Objectives
-Analyze medication sales performance
-Identify top selling products
-Evaluate revenue trends
-Detect seasonal purchasing patterns
-Geenerate actionable business insights

## Tool
-Excel

## Key Insights
-Identified the highest selling medications
-Analyzed monthly sales trends
-Evaluated product demands
-Highlighted opportunities for inventory optimization

## Outcome
The analysis provides valuable insights that can help improve inventory management, stock shortages  and support strategic business decisions.
